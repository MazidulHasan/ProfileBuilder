About Me

Senior QA Automation Engineer with 6+ years of experience delivering scalable, production-ready UI and API test automation for enterprise systems. Specialized in designing clean, maintainable automation frameworks using Playwright, Cypress, Selenium, and Rest Assured, integrated with CI/CD pipelines. Currently focused on AI-assisted testing, intelligent test optimization, and modern QA practices to improve release quality and risk assessment.

My Journey

I've grown from hands-on testing into building and scaling maintainable UI and API automation frameworks using Playwright, Cypress, Selenium, and Rest Assured, with strong CI/CD integration. At Therap (BD) Ltd, a USA-based healthcare platform, I led regression, feature, integration, and UAT testing across Agile releases. I improved API automation efficiency by 75%, increased UI test coverage by 25%, and strengthened early defect detection through shift-left testing. My experience spans healthcare and finance domains, with a strong focus on quality engineering, system reliability, and risk-based testing. I'm currently advancing toward AI-assisted testing and intelligent QA strategies.

My Approach

I apply a shift-left and risk-based quality engineering mindset, starting with requirement analysis to align testing strategy with system architecture and business objectives. I embed automation into CI/CD pipelines using Jenkins, Docker, and Git to enable fast, reliable feedback across Agile releases.

My approach combines maintainable UI and API automation, performance testing with Gatling, and structured defect management in JIRA. I work closely with distributed, cross-functional teams to drive early defect detection, maximize test coverage, and support high-confidence production releases.
